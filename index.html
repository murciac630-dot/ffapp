🔗 Conexión Google Sheets
URL de la implementación web de tu Google Apps Script
📋 Instrucciones de Configuración
1. Crea una Google Sheet en tu cuenta Gmail
2. Ve a Extensiones → Apps Script
3. Pega el código del archivo FercoFarma_AppsScript.gs que se incluye
4. Implementa → Nueva implementación → Aplicación web → Ejecutar como: Yo mismo · Acceso: Cualquier usuario
5. Copia la URL generada y pégala arriba
6. ¡Listo! Los datos se sincronizarán automáticamente con tu Google Sheet
